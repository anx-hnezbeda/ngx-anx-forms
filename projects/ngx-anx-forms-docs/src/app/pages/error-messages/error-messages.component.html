Error messages

By default the library comes with a set of messages to display errors.

Error Message
required This field is required
email Please insert a valid email
min Please insert a higher value
max 'Please insert a lower value'
*unknown Unknown error: unknown

*Any other message will be mapped as unknown error.

Custom messages

You can define custom error messages in each component instance using the [errorMessages] prop. However, you may need to define the messages for all forms and not provide them on every instance.

You can achieve this by creating a custom ErrorMessageService:


And provide the service in NgxAnxFormsModule.forRoot in the AppModule


Translate messages

You can use an external library such as ngx-translate to translate the error messages
