What is this

ngx-anx-forms is a collection of form components and form utilities for Angular.

Dependencies

This repository contains a set of native Angular forms based on Bootstrap's markup and CSS.

Library Angular Bootstrap CSS
0.1.x 6.x.x 4.1.x

Installation

You can install the library using you favorite package manager:

npm
yarn

After the instalation you have to import the main module:


And in your local module:


Load bootstrap and form styles

@import "~bootstrap/scss/bootstrap"; @import "~ngx-anx-forms/lib/styles/ngx-anx-forms.styles";

Optional: Load a form theme

@import "~ngx-anx-forms/lib/styles/themes/borderless";

Supported browsers

The library strives to support the same browsers and versions supported by Bootstrap 4 and Angular:

  • https://angular.io/guide/browser-support
  • https://getbootstrap.com/docs/4.0/getting-started/browsers-devices/
